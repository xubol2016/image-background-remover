# 植物大战僵尸程序改进建议

## 用户反馈 (xubo)

### 问题
- 僵尸数量过多，游戏难度过大

### 建议改进

1. **开局保护期**
   - 开局30秒内不要出现任何僵尸
   - 给玩家准备时间

2. **阳光系统调整**
   - 每个阳光值：25
   - 掉落频率：每30秒天空掉落一个阳光
   - 自动收集：阳光掉落1秒后自动捡取

3. **第一波僵尸**
   - 频率：每10秒出现1只
   - 类型：普通僵尸
   - 数量：共4只

4. **第二波僵尸**
   - 均匀分布
   - 组成：
     - 普通僵尸 × 3
     - 快速僵尸 × 1
     - 路障僵尸 × 2

5. **最后一波僵尸**
   - 初始共10只
   - 组成：
     - 普通僵尸 × 5
     - 快速僵尸 × 3
     - 路障僵尸 × 2
   - **特殊僵尸：举旗僵尸（BOSS）**
     - 位置：中间道路
     - 血量：普通僵尸的3倍
     - 技能：每5秒召唤1只普通僵尸
     - 召唤位置：随机出现在5个道路中的任意一个
     - 伤害参考：樱桃炸弹一次爆炸伤害 = 举旗僵尸血量的50%（需要2个樱桃炸弹才能击杀）

### 植物设计

| 植物 | 冷却时间 |
|------|----------|
| 向日葵 | 7秒 |
| 豌豆射手 | 10秒 |
| 双发射手 | 15秒 |
| 樱桃炸弹 | 20秒 |
| 大嘴花 | 7秒 |
| 坚果墙 | 7秒 |

1. **大嘴花**
   - 攻击范围：面前1.5格
   - 攻击方式：将僵尸整个吞下
   - 可吞食最大血量：普通僵尸的2倍
   - 消化时间：咀嚼1分钟后吞下
   - 樱桃炸弹伤害：普通僵尸血量的2倍

### 游戏速度设置

- 僵尸基础移动速度：降低为原来的50%（变慢一倍）
- 游戏倍速选项：1倍 / 2倍 / 3倍
